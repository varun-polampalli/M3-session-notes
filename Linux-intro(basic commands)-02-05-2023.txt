Commands :
-------------------
1.sudo su - : Is used to login as a Root user.

super user do 

2.whoami : To Display the current logged in user

3.mkdir : Is used to create a Directory.

Syntax : mkdir <directory-name>

Windows : Folders.

Linux : Directory.

4.ls : Ls is used to list the current working directory.

5.cd : Change directory.

syntax : cd <dir>

Switch to Previous directory : 

syntax : cd ..

6.pwd : Present working Directory.

7.touch : Is used to create a Files

syntax : touch <file-name>

8.Editors : vi,nano,vim

syntax : nano file-name

Save : ctrl+s
Exit : ctrl+x

9.cat : Display the content of file.

syntax : cat <file-name>

10.cp : Copy the data from one file to another or one directory another directory.

syntax : cp <Source> <destination>

syntax : cp -r <source> <destination> : Recursively
 
11.mv : To move/Transfer the data from one location to another location.

syntax : mv source destination

12.To Remove the file :

syntax : rm <file-name>

         rm -f <file-name> : Forcefully

         rm *.txt

13.rmdir : To Remove the directory.

syntax : rmdir <dir-name> : Empty Directory.

         rm -rf <dir-name> : Non empty Directory
